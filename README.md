# Thai Cafe - Premium Mock Website

A visually stunning, non-functional mock website created to showcase what a full-service website could look like for Thai Cafe restaurant in Irvine, CA.

## 🎯 Purpose

This is a **visual demo / pitch piece** designed to impress the business owner with a polished, premium, and conversion-focused web presence. All interactive elements are purely presentational - no actual functionality (payments, orders, reservations) is implemented.

## 📁 Files Included

- **index.html** - Home page with hero section, lunch specials, spice guide, photo grid, and location info
- **menu.html** - Full menu with visual categories, dietary tags, and spice indicators
- **reservations.html** - Waitlist/reservation interface with party size selector and time slots
- **order.html** - Online ordering page with customization modal and shopping cart mockup
- **gallery.html** - Photo gallery with masonry layout and category filters

## 🎨 Design System

### Color Palette
- **Primary:** #FF5722 (Bright Chili Orange)
- **Secondary:** #4CAF50 (Basil Green)
- **Accent:** #FFC107 (Turmeric Gold)
- **Neutral:** #F5F5F5 (Light Gray/Eggshell)

### Typography
- **Headings:** 'Lora' (serif) - Elegant, authentic
- **Body:** 'Outfit' (sans-serif) - Clean, modern

## 🔧 Technical Details

- **Framework:** Pure HTML/CSS/JavaScript
- **Styling:** Tailwind CSS (via CDN)
- **Fonts:** Google Fonts (Lora, Outfit)
- **Images:** High-quality placeholders from Unsplash
- **Responsive:** Fully mobile-friendly
- **Animations:** Custom CSS transitions and keyframe animations

## 🎭 Demo Mode Features

All interactive elements display a "Demo Mode" toast notification when clicked:
- Order buttons
- Cart checkout
- Reservation submission
- Social media links
- Mobile menu
- Calendar/map features

## 🚀 How to View

1. Open any HTML file in a modern web browser
2. Navigate between pages using the main navigation menu
3. All pages are standalone and can be viewed independently

**Recommended starting point:** Open `index.html` first

## 📱 Responsive Design

The website is fully responsive with breakpoints for:
- Mobile (< 768px)
- Tablet (768px - 1024px)
- Desktop (> 1024px)

## 🎪 Micro-Interactions

- **Navbar:** Shrinks and changes color on scroll
- **Menu Cards:** Pop-in animation on scroll, lift on hover
- **Spice Icons:** Glow effect on hover
- **Buttons:** Scale transform on hover
- **Cart Badge:** Bounce animation when items added
- **Time Slots:** Highlight on hover/selection
- **Gallery Images:** Zoom effect on hover, smooth lightbox transition
